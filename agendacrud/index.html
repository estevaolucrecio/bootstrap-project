Definições CRUD

Create - Inserir

Esta operação do CRUD realiza a inserção de novos itens em uma base de dados!

Read - Visualizar

Esta operação do CRUD realiza a visualização dos registros em uma base de dados!

Update - Atualizar

Esta operação do CRUD realiza a atualização dos itens em uma base de dados!

Delete - Excluir

Esta operação do CRUD realiza a exclusão de itens em uma base de dados!
